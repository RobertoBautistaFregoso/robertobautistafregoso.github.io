# Story AMA-06: Follow-up suggestion chips
As a visitor, I want suggested follow-up questions after an answer, so that I can keep exploring without having to think of what to type next.

## Acceptance Criteria
- Given an answer that includes follow-up suggestions, when it renders, then the follow-ups display as chips (buttons).
- Given I tap a follow-up chip, when I do, then it is submitted as my next question and the conversation continues.
- Given the chips, when rendered, then they are keyboard-reachable with visible focus.

## Non-goals
- Generating the follow-ups (that's the agent/backend).
- Analytics on chip usage (Measure / v2).
