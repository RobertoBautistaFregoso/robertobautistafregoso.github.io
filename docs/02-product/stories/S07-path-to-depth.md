# Story S07: Path to depth (navigate to resume)
As an interested visitor, I want to navigate from the home page to Roberto's full resume, so that I can dig into his experience and trajectory.

## Acceptance Criteria
- Given the home page, when I look for more detail, then there is a clearly labeled link/nav item pointing to the resume.
- Given that link, when I activate it, then I arrive at the resume page.
- Given the resume page has not been built yet (this story may land before the resume slice), when I activate the link, then it resolves to an intentional placeholder — never a 404 or broken link — tracked as a follow-up.

## Non-goals
- The resume page content itself (that's the next slice: the Resume screen)
- Multi-level navigation or a full site menu beyond what v0's 2 screens need
