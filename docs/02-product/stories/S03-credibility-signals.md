# Story S03: Credibility signals
As a visitor, I want to see the industries Roberto has worked in, the clients/brands he's worked with, and the types of projects he's done, so that I can quickly gauge his credibility and relevance.

## Acceptance Criteria
- Given the home page, when I scroll past the hero, then I see a section listing industries/domains of experience.
- Given that section, when it renders, then it shows client/brand names or logos he has worked with.
- Given that section, when I read it, then it lists the types of projects he has participated in.
- Given any client/brand shown, when it is displayed, then it respects NDA constraints — only publicly-showable names/logos appear; others are anonymized (e.g., "a Fortune-500 fintech") or omitted.

## Non-goals
- Detailed per-client case studies (V2)
- Logo-wall animations or interactivity
